## 删除最外层括号问题

* 删除最外层括号，输出内层括号，类似的："(())((()))"，我们可以看作"("+")"的形式，就可以写成"("**()**")"+"("**(())**")"=**()(())**，加粗部分就是我们得到的结果，例子如下：

例1：
```
输入：(()())(())
输出：()()()
```

例2：
```
输入：()()
输出：""
//因为"()"为字符串，如果结果没有，就输出空的""表示
```
具体操作如下：

1. 我们需要两个整形变量进行计数，还需要定义一个字符串变量

    a. 两个整形变量初始化时一个为"1"，另一个为"0"，我们可以通过这样定义来区别最外层括号和内层括号

2. 使用循环进行遍历，遍历时，需要从1开始

    a. 筛选出左括号和右括号，当为分别为左右括号时，计数器都需要分别计数

    b. 筛选出我们要的结果
        
        (1). 当两个计数器不相同时，我们需要把这个括号放到定义的字符串
        中，用来保存结果

        (2). 当计数器相同时，说明找到最外层的括号，我们需要重新计数，
        重新把计数器赋值成初始化相同的数，循环也需要再加一次(不然当有
        第二个外层括号时，我们无法删除第二个外层括号)

3. 返回字符串结果
