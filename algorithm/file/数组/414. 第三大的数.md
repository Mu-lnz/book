## 第三大的数

在一组数组中，找出第三大的数，如果没有这个数，就返回最大的那个数，要求时间复杂度为O(n)

例1：
```
输入：
    3 2 1
输出：
    1
```
例2：
```
输入：
    1 2 2
输出：
    2
```

具体操作：

1. 可以直接定义一个集合类型set，set可以直接去重和排序

2. 当set的大小小于3时，返回最后一个元素

3. 否则返回第三个元素
