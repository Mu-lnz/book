## 编写c++程序
### 1. 一个c++程序包含一个或多个函数，其中一个的函数名必须为main，一个函数编写的方式如下：
```c++
int find(int x){
    return 0;
}
```
从上可知，一个函数的编写包括**返回类型**、**函数名**、**形参列表**和**函数体**<br>
在find这个函数中，返回类型为int，函数名称为find，参数列表就是()<br>
内容，函数体是{}中包含的内容<br>
函数体通常是以左括号"{"开始，右括号"}"结束

### 2. 运行程序

在PC机中，我们可以在网上搜寻C++的编译器，其中最广为人知的编译器有**Visual Studio**这类编译器，当然，如果只是用于学习，那么使用**dev**这类编译器就可以了<br>

现在PC机上的编译器大多都具有集成开发环境，集成开发环境是与其他程序创建和分析工具包装在一起的编译器<br>

在编译一个c++程序时，我们的c++文件后缀名通常是 **.cpp**，编译一个c++文件后，系统会生成一个 **.exe**文件，如果是使用linux系统，那么编译后，会生成一个 **a.out**文件
